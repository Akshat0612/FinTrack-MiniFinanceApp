# FinTrack – Personal Finance Management System

## Overview

FinTrack is a personal finance management application designed to help users track transactions, monitor investments, manage liabilities, and forecast future portfolio growth through a unified dashboard.

The application combines financial analytics, portfolio tracking, SIP projections, liability planning, CSV-based portfolio import/export, and interactive visualizations to support informed financial decision-making.

## Features

### Transaction Management

* Track income and expenses
* Categorize transactions
* Search, sort, and filter records
* Monthly and yearly financial analysis
* Export transaction summaries as CSV

### Investment Portfolio Tracking

* Monitor mutual fund holdings
* Analyze portfolio allocation
* Calculate current portfolio value
* Track fund-wise investment performance

### SIP Forecasting

* Estimate future portfolio value based on monthly SIP contributions
* Allocation-based growth projections
* Category-wise return assumptions

### Liability Management

* Track upcoming liabilities and due dates
* Prioritize financial obligations
* Assess liability coverage using available funds and investments
* Manage planned future income sources

### Data Visualization

* Expense distribution charts
* Income and expense analytics
* Portfolio allocation insights
* Interactive financial dashboards

### Additional Features

* CSV-based mutual fund portfolio import
* CSV export for financial summaries
* Light/Dark mode support
* Persistent local storage
